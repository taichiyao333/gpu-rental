📖 セットアップガイドを見る

📖 GPU貸し出し はじめからセットアップ

1

NVIDIAドライバをインストール

nvidia.com/drivers から最新ドライバをダウンロードしてインストール。
インストール後に再起動が必要です。

2

Node.js 20 をインストール

nodejs.org からLTS版をダウンロード。
またはコマンド: winget install OpenJS.NodeJS.LTS

3

GPURentalエージェントを起動

プロバイダーポータルからエージェントをダウンロードして実行してください。
または: npm start

4

GPUを登録する

エージェント起動後、プロバイダーポータルからGPUを登録すれば完了です。
