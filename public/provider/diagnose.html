📖 GPU貸し出し はじめからセットアップ

1

NVIDIAドライバをインストール

nvidia.com/drivers から最新ドライバをダウンロードしてインストール。
インストール後に再起動が必要です。

2

Node.js 20 をインストール

nodejs.org からLTS版をダウンロード。
またはコマンド: winget install OpenJS.NodeJS.LTS

3

GPURentalエージェントをインストール・起動

💡 GPURentalエージェントとは？
あなたのPCにインストールする常駐プログラムです。
GPU情報をリアルタイムでGPURentalサーバーに送信し、ユーザーがあなたのGPUを利用できる状態に保ちます。
📌 ポートフォワーディング不要 · バックグラウンド動作 · 自動再接続
① エージェントをダウンロード
🔗 プロバイダーポータル → 「エージェントをダウンロード」ボタン
📥 ポータルへ行く
② PowerShellで実行（管理者パワーシェル）
cd gpu-rental-agent
npm install
npm start
① エージェントをクローン
git clone https://github.com/gpurental/agent.git
cd agent && npm install
npm start
🔌 エージェント起動状態
「確認する」を押すとエージェントの起動状態をチェックします
4

GPUを登録する

エージェント起動後、プロバイダーポータルからGPUを登録すれば完了です。
