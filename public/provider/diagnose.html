GPU貸し出しの準備チェック

ボタンを押すと、あなたのPCでGPU貸し出しに必要な環境が
すべて整っているか自動的に確認します。
問題があれば、その場で解決手順をご案内します。

📖 セットアップガイドを見る

📖 GPU貸し出し はじめからセットアップ

1

NVIDIAドライバをインストール

nvidia.com/drivers から最新ドライバをダウンロードしてインストール。
インストール後に再起動が必要です。

2

Node.js をインストール

💡 Node.jsとは？
GPURentalサーバーを動かすために必要な実行環境です。一度インストールすれば次回以降は不要です。
1 Node.js をダウンロードする
🟢 Node.js 20 LTS（推奨版）
「LTS」と書いてある安定版を選んでください
📥 nodejs.org を開く
サイトを開いたら「LTS 推奨版」と書かれたボタンをクリックしてダウンロード
2 インストーラーを実行する
→ ダウンロードした node-v20.x.x-x64.msi をダブルクリック
→ 「Next」を繰り返し押して進む（設定は変えなくてOK）
→ 「Install」をクリック → インストール完了
→ インストール後、PCを再起動してください
💡「このアプリが変更を加えることを許可しますか？」→「はい」をクリック
3 インストールを確認する
PowerShellを開いて（管理者不要）、以下のコマンドを実行：
node --version
✅ v20.x.x のようにバージョンが表示されれば成功です！
「認識されていない」と出た場合は、PCを再起動してから再度お試しください。
3

GPURentalエージェントをインストール・起動

💡 GPURentalエージェントとは？
あなたのPCにインストールする常駐プログラムです。
GPU情報をリアルタイムでGPURentalサーバーに送信し、ユーザーがあなたのGPUを利用できる状態に保ちます。
📌 ポートフォワーディング不要 · バックグラウンド動作 · 自動再接続
1 ZIPファイルをダウンロードする
📦 GPURental Server（ZIP）
ボタンをクリックしてZIPをダウンロード
📥 ZIPでダウンロード
📂 展開先はここがおすすめ：
ダウンロードしたZIPを右クリック →「すべて展開」→ 展開先を C:\ に変更 → 展開
これで C:\gpu-rental-main というフォルダができます ✅
2 PowerShellを「管理者として」開く
→ Windowsキーを押して「PowerShell」と入力
→ 表示された「Windows PowerShell」を右クリック
→ 「管理者として実行」を選択 → 青いウィンドウが開く
💡「このアプリにデバイスに変更を加えることを許可しますか？」が出たら「はい」をクリック
3 PowerShellで以下のコマンドを順に実行
まずフォルダを移動します（コピーして貼り付けてEnter）：
cd C:\gpu-rental-main
必要なライブラリをインストールします（少し時間がかかります）：
npm install
サーバーを起動します：
npm start
✅ PowerShellに Server running at http://localhost:3000 と表示されれば起動成功です！
このウィンドウは閉じないでください。閉じるとサーバーが停止します。
① リポジトリをクローン
git clone https://github.com/taichiyao333/gpu-rental.git
cd gpu-rental && npm install
npm start
🔌 エージェント起動状態
「確認する」を押すとエージェントの起動状態をチェックします
4

GPUを登録する

エージェント起動後、プロバイダーポータルからGPUを登録すれば完了です。
