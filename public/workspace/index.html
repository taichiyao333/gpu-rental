VS Code

VS Code Remote SSH

  1. VS Code に Remote - SSH 拡張機能をインストール
    → インストールページを開く
  2. ~/.ssh/config に以下を追記： 
    Host gpu-rental
    HostName (上のSSHホスト)
    Port 2222
    User gpu-user-(ユーザーID)
  3. Ctrl+Shift+P → Remote-SSH: Connect to Host → gpu-rental を選択
  4. パスワードを入力してリモートGPUマシンに接続
  5. ファイル → フォルダを開く → 作業ディレクトリを指定
✅ 接続後はVS Codeのすべての機能（デバッグ、拡張機能、Git等）がリモートGPU上で動作します
Jupyter

JupyterLab（機械学習・データ分析）

  1. 上のターミナルタブで以下を実行： 
    pip install jupyterlab -q
jupyter lab --ip=0.0.0.0 --port=8888 --no-browser --NotebookApp.token='gpurental'
  2. ブラウザで以下にアクセス（新しいタブ）：
    読み込み中...
⚠️ JupyterLabポートはCloudflareトンネル経由で公開されていません。SSHポートフォワードを使うか、ブラウザ端末で直接利用してください
PyCharm

PyCharm Remote Interpreter

  1. File → Settings → Project → Python Interpreter
  2. Add → SSH Interpreter
  3. SSH接続情報（上記）を入力して接続
  4. Tools → Deployment → Configuration でワークスペースパスを紐付け

⚠️ 注意事項

  • セッション終了時間になると自動切断されます（10分前にメール警告）
  • /workspace 内のファイルは次回接続時も保持されます
  • CUDAは利用可能。torch・tensorflowはデフォルトでインストール済みです
  • クリックするとコード/コマンドをコピーできます

3サイドバー(N) → GPURental タブからログイン
4GPUを選択して「☁ クラウドレンダリング開始」
